# Code for plotting pole figures of Bicrystal Orientation
## By Dr Julian Mecklenburgh University of Manchester
The code uses [MatLab](https://matlab.mathworks.com/) and the toolbox [MTex](https://mtex-toolbox.github.io/index). The scripts have used version 6.1.0 of MTex.

Scripts used to plot figures and calculate Schmid factors in Pratap-Singh et al. (2026) 
Bicystal_polefigure.m plots pole figure of the bicrystal orientation and the compression and shear direction.
Forsterite_slip_systems.m calculates the Schmid Factors $m'$ factor and angles between boundary and slip plane.


## References
Pratap Singh,S., Mariani, E., Mecklenburgh,J., Thom, C., , Marquardt, K., Wheeler,J., and Hansen, L. (2026), Direct Measurement of Grain-Boundary Sliding in Forsterite Bicrystals.
